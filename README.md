# Plugin Manager Template

![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/Smertan/plugin-manager/ci.yml)

A `cargo-generate` template for creating flexible and easy-to-use plugin management systems in Rust applications.

## What is this template?

This template provides a complete plugin architecture that enables dynamic loading, registration, and management of plugins at runtime. It supports both individual plugins and grouped plugins, making it suitable for various application architectures.

## Prerequisites

- Rust (latest stable version recommended)
- `cargo-generate` installed:

    ```bash
    cargo install cargo-generate
    ```
